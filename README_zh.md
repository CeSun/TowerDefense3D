# Tower Defense 3D

[English](README.md)

基于 [Avalonia UI](https://avaloniaui.net/) 和 [Aura3D](https://github.com/CeSun/Aura3D) 构建的 3D 塔防游戏。

在敌人路径旁放置防御塔，阻止一波波敌人抵达终点。管理金币、搭配塔型，挺过全部波次。

![截图](screenshot.png)

## 特性

- 7 种防御塔，各有独特机制（单体、溅射、减速、多重箭、暴击、持续毒伤、范围灼烧）
- 3 种敌人（基础、快速、重装），每张地图 5+ 波进攻
- **内置地图编辑器** — 实时 3D 预览设计自定义地图，配置路径与波次，一键试玩
- 10 张预置地图，难度递进，自动保存玩家进度
- 等轴测 3D 视角，实时战斗、弹道和血条
- 跨平台：Windows、macOS、Linux & Android

## 技术栈

| 层级 | 技术 |

## 快速开始

### 环境要求

- [.NET 10.0 SDK](https://dotnet.microsoft.com/download)

### 运行（桌面端）

```bash
cd TowerDefense.Desktop
dotnet run
```

### 构建

```bash
dotnet build
```

## 玩法

1. 在关卡选择界面选择一个关卡
2. 点击右侧面板的**塔按钮**选择塔类型
3. 点击地图上的**绿色格子**放置防御塔（棕色路径格不可建造）
4. 防御塔自动攻击范围内的敌人
5. 守住全部波次即为胜利；20 条命耗尽则游戏结束

### HUD 说明

| 元素 | 描述 |
|---|---|
| 金币 | 击杀敌人获得；用于建造防御塔 |
| 生命 | 敌人到达终点时减少 |
| 波次 | 当前波数 / 总波数 |
| 状态 | 上下文提示和游戏信息 |

## 地图编辑器

内置地图编辑器支持实时 3D 预览，可创建和修改地图：

- **网格尺寸** — 可调整列数和行数
- **路径点** — 点击格子设置起点/终点和路径拐点
- **波次配置** — 添加波次，设置敌人类型、数量和生成间隔
- **测试** — 从编辑器直接试玩地图
- **保存** — 地图以 JSON 文件存储在 `Maps/` 目录

## 地图

- 10 张预置关卡，难度递增
- 自动保存玩家进度（最高解锁关卡）
- 地图为 JSON 文件，可手动编辑或分享
