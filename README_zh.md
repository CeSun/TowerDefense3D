# Tower Defense 3D

[English](README.md)

基于 [Avalonia UI](https://avaloniaui.net/) 和 [Aura3D](https://github.com/CeSun/Aura3D) 构建的 3D 塔防游戏。

在蜿蜒的路径上放置防御塔，阻止一波波敌人抵达终点。管理金币、搭配塔型，挺过全部 5 波进攻。

![截图](screenshot.png)

## 技术栈

| 层级 | 技术 |
|---|---|
| UI 框架 | Avalonia 12 + Fluent 主题 |
| 3D 引擎 | Aura3D (Avalonia 集成) |
| .NET | net10.0, C# |
| 平台 | Windows Desktop, Android |

## 项目结构

```
TowerDefense3D/
├── TowerDefense/              # 核心游戏库
│   ├── GameData.cs            # 塔/敌人定义，运行时数据
│   ├── GameManager.cs         # 核心逻辑：波次、战斗、寻路
│   ├── GameView.axaml         # UI 布局（HUD、塔面板、覆盖层）
│   ├── GameView.axaml.cs      # 3D 场景、输入、模型构建、UI 同步
│   ├── App.axaml / .cs        # Avalonia 应用入口
│   └── MainWindow.axaml / .cs # 主窗口
├── TowerDefense.Desktop/      # Windows 桌面启动器
├── TowerDefense.Android/      # Android 启动器
└── TowerDefense3D.slnx        # 解决方案文件
```

## 快速开始

### 环境要求
- [.NET 10.0 SDK](https://dotnet.microsoft.com/download)

### 运行（桌面端）

```bash
cd TowerDefense.Desktop
dotnet run
```

### 构建

```bash
dotnet build
```

## 玩法

1. 点击 **Start Game** 开始第一波
2. 点击右侧 **塔按钮** 选择塔类型
3. 点击地图上的 **绿色格子** 放置防御塔（棕色路径格不可建造）
4. 防御塔自动攻击范围内的敌人
5. 守住全部 5 波即为胜利；20 条命耗尽则游戏结束

### HUD 说明

| 元素 | 描述 |
|---|---|
| 💰 金币 | 击杀敌人获得；用于建造防御塔 |
| ❤️ 生命 | 敌人到达终点时减少 |
| 🌊 波次 | 当前波数 / 总波数 |
| 状态 | 上下文提示和游戏信息 |

## 防御塔

### 基础塔

| 塔 | 价格 | 伤害 | 射程 | 攻速 | 特性 |
|---|---|---|---|---|---|
| 🏹 弓箭塔 | 50g | 15 | 3.5 | 1.8/秒 | 快速廉价单体输出 |
| 💣 炮塔 | 100g | 40 | 3.0 | 0.6/秒 | 溅射半径 1.5 |
| ❄️ 冰塔 | 75g | 8 | 2.8 | 1.0/秒 | 减速敌人 50% |

### 高级塔

| 塔 | 价格 | 伤害 | 射程 | 攻速 | 特性 |
|---|---|---|---|---|---|
| 🧨 多重箭 | 150g | 12×3 | 3.0 | 1.0/秒 | 扇形发射 3 箭，各追踪不同敌人 |
| 🎯 狙击塔 | 130g | 50 | 5.0 | 0.5/秒 | 30% 概率暴击 ×3 伤害；射程最远 |
| ☠️ 毒针塔 | 100g | 8 | 3.5 | 1.5/秒 | 命中附带持续毒伤：20/秒，持续 3 秒 |
| ☀️ 太阳塔 | 175g | 18/秒 | 2.5 | — | 持续灼烧周围敌人（无弹道） |

## 地图

- **20×12 网格**，每格 1 单位
- 棕色路径从左至右蜿蜒穿过全图
- 绿色格子可建造
- 敌人从 **左侧**（绿色标记）生成，抵达 **右侧**（红色标记）即扣命
